# Canonical Designs

## Статус

В проекте `scoring` зафиксирован каноничный комплект из трех UI-экранов.

Этот набор считается baseline для следующих экранов и следующих UI-итераций, пока отдельно не согласован новый design-pass.

## Каноничный комплект

### 1. Home

Файл:

- [prototype-home.html](/C:/Users/illki/Desktop/projects/scoring/prototype-home.html)

Назначение:

- главный экран;
- реестр проектов;
- базовый визуальный язык продукта.

Что из него наследуется:

- шапка;
- общий ритм страницы;
- карточки;
- фильтры;
- footer.

### 2. Month

Файл:

- [prototype-month.html](/C:/Users/illki/Desktop/projects/scoring/prototype-month.html)

Назначение:

- месячный слой;
- архивный срез;
- промежуточная навигация между реестром и деталкой.

Что из него наследуется:

- представление месяца;
- компактная статусная статистика;
- логика перехода в конкретные проекты.

### 3. Detail

Файл:

- [prototype-detail.html](/C:/Users/illki/Desktop/projects/scoring/prototype-detail.html)

Назначение:

- детальная карточка проекта;
- рабочий экран для редактирования и просмотра;
- эталон для form-layout и правой колонки ссылок.

Что из него наследуется:

- структура секций;
- pattern полей;
- pattern редактируемых ссылок;
- календарные поля;
- side rail и stage/doc blocks.

## Правило использования

Новые страницы в `scoring` должны:

- либо продолжать один из этих трех канонов;
- либо явно вводить новый дизайн-pass с обновлением [UI-REFERENCE.md](/C:/Users/illki/Desktop/projects/scoring/docs/UI-REFERENCE.md) и этого документа.
