## Agent Loop

1. Прочитать `AGENTS.md`.
2. Прочитать этот sprint-файл.
3. Открыть только выбранную задачу через `backlog task view TASK-N --plain`.
4. Если задача меняет поведение продукта, проверить или создать связанный OpenSpec change.
5. Перед началом работы перевести задачу в `In Progress`.
6. После реализации обновить task notes/final summary и статус.

## Commands

```powershell
backlog task list --plain
backlog task view TASK-1 --plain
backlog task edit TASK-1 -s "In Progress"
backlog task edit TASK-1 --notes "Implementation notes here"
backlog task edit TASK-1 --final-summary "Final result here"
backlog task edit TASK-1 -s "Done"
backlog browser --no-open --port 6420
```
